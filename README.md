# Correr el proyecto localmente

1. Definir en controllers/auth/auth.controllers.js si se quiere ejecutar como admin o no
```js
const admin = true | false; // Por defecto es true
```

2. Ejecutar el archivo index.js:
  * Usando node: 
```
node index.js 
```
  * Usando nodemon: 
```
nodemon index.js 
```
