# Auth mock config

### Definir en `controllers/auth/auth.controllers.js` si se quiere ejecutar como admin o no

```js
const admin = true | false; // Por defecto es true
```

# Correr el proyecto localmente

### Ejecutar el archivo `index.js`:

-   Usando node:

```
node index.js
```

-   Usando nodemon:

```
nodemon index.js
```
